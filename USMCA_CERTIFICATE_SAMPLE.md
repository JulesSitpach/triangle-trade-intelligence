### Field 7: Origin Criterion

**Which rule makes this product qualify for USMCA?**

```
â˜‘ A - Wholly obtained or produced entirely in USMCA territory
â˜ B - Produced entirely from originating materials
â˜ C - Satisfies specific rule of origin (tariff shift + RVC)
â˜ D - Produced entirely in USMCA territory from non-originating materials
```

**For TechFlow:**
- âœ… Check "C" because:
  - Product has Chinese components (35% non-originating)
  - BUT satisfies RVC requirement (65% regional content achieved)
  - Tariff shift occurred (components changed classification during manufacturing)

**How to know:**
- A = 100% USMCA materials (rare)
- B = All materials already qualified as USMCA (uncommon)
- C = **Most common** - has non-USMCA materials but meets RVC threshold
- D = Auto/textile specific rules

### âŒ Wrong HS Code
**Mistake:** Using component HS codes (8542.31.00 for microprocessor)

### âŒ Wrong Country of Origin
